📋 Sıradaki Adımlar:

  1. Immich'i aç: http://localhost:2283
  2. Admin hesap oluştur (email + şifre)
  3. Settings → Account → API Keys
  4. "Create New Key" → Token'i kopyala
  5. Buraya dön ve API Key'i güncelle
  6. n8n: http://localhost:5678
  7. Workflow'ları import et (n8n-workflows/01_poll_and_route.json)
  8. Mobil: Immich App indir ve server ekle

💡 İpucu: Immich mobil uygulamasını indirdikten sonra, telefondan çektiğin fotoğraflar otomatik olarak yüklenir ve sisteme girer.
